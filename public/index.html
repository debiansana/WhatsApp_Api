Cara koneksikan VPN Remote pada Mikrotik

Script L2TP :

:if ([:len [/ppp profile find name="server.vpn"]]=0) do={/ppp profile add name="server.vpn" comment="server.vpn"};/interface l2tp-client add user="uservpn" password="passvpn" connect-to="server.vpn" profile=server.vpn name="l2tp-server.vpn-uservpn" keepalive-timeout=10 use-peer-dns=no disabled=no comment="server.vpn tunnel remote L2TP";

Script PPTP :

:if ([:len [/ppp profile find name="server.vpn"]]=0) do={/ppp profile add name="server.vpn" comment="server.vpn"};/interface pptp-client add user="uservpn" password="passvpn" connect-to="server.vpn" profile=server.vpn name="pptp-server.vpn-uservpn" keepalive-timeout=10 use-peer-dns=no disabled=no comment="server.vpn tunnel remote PPTP";

Script WhatsApp BOT Monitoring Mikrotik

Script PPPoE :

On Up / Login
:local nama "$user";
:local ips [/ppp active get [find name=$nama] address];
:local up [/ppp active get [find name=$nama] uptime];
:local caller [/ppp active get [find name=$nama] caller-id];
:local service [/ppp active get [find name=$nama] service];
:local active [/ppp active print count];
:local profile [/ppp secret get [find name=$nama] profile];
:local datetime "Tanggal: $[/system clock get date] %0AJam: $[/system clock get time]";
